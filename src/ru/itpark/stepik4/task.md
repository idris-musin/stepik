## 2.1 Примитивные типы

Реализуйте метод <convert style=color:red> flipBit </convert>, изменяющий значение одного бита заданного целого числа на противоположное. Данная задача актуальна, например, при работе с битовыми полями.

Договоримся, что биты нумеруются от младшего (индекс 1) к старшему (индекс 32).

Воспользуйтесь предоставленным шаблоном. Декларацию класса, метод main и обработку ввода-вывода добавит проверяющая система.

```markdown
Sample Input:
0 1

Sample Output:
1
```
